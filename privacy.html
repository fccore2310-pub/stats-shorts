Privacy Policy

Last updated: April 29, 2026

FC Core Publisher ("the Tool") is a personal automation script. This policy describes how data is handled.

1. Who runs the Tool

The Tool is operated by a single individual (the Operator) for the sole purpose of publishing the Operator's own video content to social platforms (TikTok, YouTube, Instagram) owned by the Operator.

2. Data we collect

The Tool only handles data belonging to the Operator's own accounts:

  • OAuth access tokens issued by social platforms to the Operator
  • Public metadata (follower counts, view counts, video IDs) of the Operator's own accounts
  • Video files created by the Operator

The Tool does not collect, process or store data of any third-party user. It does not interact with users other than the Operator.

3. Storage

OAuth tokens are stored locally on the Operator's machine and on GitHub Actions encrypted secrets. They are never shared, sold, or transmitted to any party other than the social platform that issued them.

4. Third-party services

The Tool calls APIs of TikTok, YouTube (Google), Meta (Instagram), and Anthropic Claude API for metadata generation. Each of those services has its own privacy policy, which applies to data exchanged with them.

5. Data retention

OAuth tokens are kept until the Operator revokes them. Analytics snapshots are kept for historical comparison and contain only the Operator's own public metrics.

6. Your rights

As the Operator is the only data subject, the Operator has full control to delete tokens, revoke access from each platform's settings, and delete any local or remote storage at any time.
